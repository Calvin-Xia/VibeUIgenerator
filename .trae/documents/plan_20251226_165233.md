## UI 组件代码导出功能模块设计方案

### 现有代码分析

目前项目中已存在部分导出基础设施：
- `components/output/` 目录包含 `OutputPanel`、`ExportButtons`、`CodeBlock`、`CodeTabs`
- `lib/generator/index.ts` 包含 `generateCSS`、`generateTailwindLayer`、`generateHTMLSnippets` 函数

### 功能规划

#### 1. 新增代码生成函数（`lib/generator/export.ts`）

**React 组件导出**：
- 生成完整的 React 组件文件
- 支持 inline styles 和 CSS modules 两种模式
- 包含 TypeScript 类型定义

**Vue 组件导出**：
- 生成 Vue 3 SFC 组件
- 支持 Composition API
- 包含 Props 类型定义

**增强 CSS 导出**：
- 生成完整的 CSS 代码块
- 包含所有必要的样式属性
- 支持 CSS 变量版本

**JSON Tokens 导出**：
- 导出设计 tokens 为 JSON 格式
- 便于其他工具或框架使用

#### 2. 增强导出面板组件（`components/output/`）

**CodePreview 组件**：
- 语法高亮显示
- 实时预览生成的代码
- 支持多种代码格式切换

**ExportModal 组件**：
- 弹窗式导出界面
- 格式选择器（React/Vue/HTML/Tailwind/CSS/JSON）
- 代码预览区域
- 复制和下载按钮

**DownloadButton 组件**：
- 下载代码为文件
- 支持自定义文件名

#### 3. 用户体验优化

- 复制成功提示（Toast 通知）
- 下载进度反馈
- 错误状态显示
- 响应式设计

### 文件结构

```
lib/generator/
  └── export.ts          # 新增：代码导出生成函数

components/output/
  ├── ExportModal.tsx    # 新增：导出弹窗
  ├── CodePreview.tsx    # 新增：代码预览组件
  ├── ExportPanel.tsx    # 增强：主导出面板
  └── ExportButtons.tsx  # 增强：导出按钮组

components/ui/
  └── index.ts           # 导出新组件
```

### 实现步骤

1. 创建 `lib/generator/export.ts` - 实现所有代码生成函数
2. 创建 `components/output/ExportModal.tsx` - 导出弹窗组件
3. 创建 `components/output/CodePreview.tsx` - 代码预览组件
4. 增强现有 `OutputPanel.tsx` - 集成新功能
5. 更新 `ExportButtons.tsx` - 添加更多导出选项
6. 测试所有导出功能

### 预期效果

- 支持 6 种代码格式导出（React、Vue、HTML、Tailwind、CSS、JSON）
- 实时预览生成的代码
- 一键复制和下载
- 良好的用户体验和错误处理
