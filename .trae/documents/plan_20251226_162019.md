## Base Radius 滑动条无效问题修复方案

### 问题分析

通过代码分析发现：

1. **InspectorPanel 第 132 行**：Base Radius 滑动条绑定的是 `tokens.theme.radius.baseRadius`

2. **getButtonStyles 第 424 行**：按钮使用的是 `button.radius`，不是 `baseRadius`

3. **getCardStyles 第 454 行**：卡片使用的是 `card.radius`，不是 `baseRadius`

4. **数据分离**：这三个值是独立的：
   - `baseRadius` 默认 12px（主题配置）
   - `button.radius` 默认 8px（按钮配置）
   - `card.radius` 默认 16px（卡片配置）

**问题根源**：Base Radius 滑动条更新的是 `theme.radius.baseRadius`，但演示界面使用的是 `button.radius` 和 `card.radius`，两者没有关联。

### 修复方案

修改 `vibeStore.ts` 中的 `setToken` 函数，当设置 `theme.radius.baseRadius` 时，自动同步更新 `button.radius` 和 `card.radius`：

1. **在 setToken 中添加联动逻辑**：
   - 当 path 是 `theme.radius.baseRadius` 时
   - 同时更新 `button.radius` 和 `card.radius`

2. **确保版本号递增**：保持版本变化触发 UI 更新

### 修改文件
- `lib/store/vibeStore.ts`

### 预期效果
- 拖动 Base Radius 滑动条时，按钮和卡片的圆角同时变化
- 演示界面实时反映圆角数值变化
