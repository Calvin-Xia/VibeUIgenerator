## 实施计划：代码语法高亮功能

### 方案概述

使用项目中已有的 `shiki` 库实现语法高亮，支持 React、Vue、HTML、Tailwind、CSS、JSON 等语言。

### 实施步骤

1. **创建语法高亮工具函数** (`lib/generator/highlight.ts`)

   * 初始化 shiki 高亮器

   * 提供 `highlightCode(code, language)` 函数

   * 支持的语言映射：typescript → tsx, javascript → ts, css, html, json, vue

2. **创建 SyntaxHighlighter 组件** (`components/output/SyntaxHighlighter.tsx`)

   * 接收 code 和 language 属性

   * 使用 useMemo 缓存高亮结果

   * 返回带有 HTML 的 div 元素

   * 支持服务端渲染避免 hydration 不匹配

3. **更新 EnhancedCode 组件** (`components/output/EnhancedCode.tsx`)

   * 替换原生 `<code>` 为 SyntaxHighlighter 组件

   * 保持行号和滚动功能

4. **应用行业标准配色方案**

   若处于亮色主题，使用vscode中light主题配色：

   若在暗色主题下，使用 VS Code 暗色主题配色

   * 关键字：粉红/紫色

   * 字符串：绿色/橙色

   * 注释：灰色

   * 函数名：蓝色

   * 变量：青色

   * JSON 键：黄色
