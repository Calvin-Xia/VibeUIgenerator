## Palette 色盘选项卡展开问题的修复方案

### 问题分析

检查了 `ColorPickerRow.tsx` 和 `InspectorPanel.tsx` 后，发现以下可能导致"展开不完全"的问题：

1. **Accordion 默认展开状态** - InspectorPanel 使用 `defaultValue={['theme', 'effects', 'button']}`，但如果 persist 恢复后状态丢失，可能导致某些 accordion item 不展开

2. **Radix UI Accordion Content 可能被裁剪** - 组件没有使用 Radix 的样式，可能导致内容显示不完整

3. **颜色选择器弹窗可能被裁剪** - `overflow-hidden` 父容器可能裁剪颜色选择器的弹出层

### 修复方案

1. **检查并修复 InspectorPanel 的 overflow 样式**
   - 确保 flex 容器不会裁剪 accordion 内容

2. **为 Accordion 添加 Radix 样式**
   - 使用 `@radix-ui/react-accordion` 提供的样式类确保内容正确展开

3. **验证颜色选择器组件**
   - 确保弹窗不会被父容器裁剪

4. **测试 Theme → Palette 下的所有颜色选择器**
   - 验证 6 个颜色选择器（Accent, Background, Surface, Text, Muted Text, Border）都能正常展开和使用
