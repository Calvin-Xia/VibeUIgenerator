## Glow 效果滑动条无变化问题修复方案

### 问题分析

通过代码检查发现 Glow 效果的问题：

1. **CanvasBackground 光晕大小和透明度是硬编码的**（第 39-51 行）：
   - `h-40 w-40` 和 `h-32 w-32` 固定尺寸，没有使用 `effects.glow.size`
   - `blur-3xl` 固定模糊值，没有使用动态模糊计算
   - 透明度使用固定的 `40`（rgba 中的 0.4）和 `30`（0.3），没有使用 `effects.glow.opacity`

2. **generateGlowShadow 函数存在但未被使用**：
   - `shadow.ts` 中有 `generateGlowShadow(size, opacity, color)` 函数
   - 但 CanvasBackground 没有调用它，直接硬编码了光晕样式

### 修复方案

修改 `CanvasBackground.tsx`：

1. **动态计算光晕尺寸**：
   - 使用 `effects.glow.size` 动态设置光晕大小
   - 计算模糊半径 = size * 2 确保视觉一致

2. **动态应用透明度**：
   - 使用 `effects.glow.opacity` 控制光晕强度
   - 将 opacity 转换为 rgba 中的 alpha 值（opacity * 0.01）

3. **保持动画效果**：
   - 保留 `animate-pulse-glow` 动画
   - 确保光晕实时响应滑动条变化

### 修改文件
- `components/preview/CanvasBackground.tsx`

### 预期效果
- 启用 Glow 并勾选后，拖动 Size 滑动条，光晕大小实时变化
- 拖动 Opacity 滑动条，光晕透明度实时变化
- 光晕效果与应用的主题 accent 颜色保持一致
