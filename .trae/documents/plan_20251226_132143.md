## 实现里程碑

### Milestone 1: 项目骨架与配置
- package.json 依赖安装
- Next.js 14 配置
- TailwindCSS + shadcn/ui 配置
- ESLint + Prettier 配置

### Milestone 2: 核心类型与 Generator 引擎
- Tokens 数据结构定义
- 颜色工具函数
- 阴影生成算法
- CSS 生成引擎

### Milestone 3: 状态管理
- Zustand store 实现
- LocalStorage 持久化
- URL 编码/解码

### Milestone 4: 12+ 内置预设
- glass, neo-brutal, soft-shadow, cyber, y2k, aurora, mono, retro, noir, pastel, clay, paper

### Milestone 5: UI 组件
- AppShell, TopBar, MainLayout
- Inspector Panel (颜色/滑杆/开关)
- Preview Canvas (背景/噪点/光斑)
- Output Panel (代码高亮/导出)

### Milestone 6: 分享与导出
- URL 分享功能
- PNG 导出
- JSON 导入

## 关键技术实现

1. **样式生成引擎**: 纯函数 tokens -> { cssVars, cssText, tailwindLayer }
2. **阴影算法**: elevation -> 多层 box-shadow
3. **玻璃效果**: backdrop-filter + 半透明背景
4. **对比度计算**: WCAG 2.1 标准
5. **URL 压缩**: lz-string 编码
