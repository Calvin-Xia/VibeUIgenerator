## 修复计划：确保代码导出按钮可见且功能正常

### 问题诊断
- `ExportButtons` 只在 Code 面板中显示，需要用户手动切换标签页
- 移动端布局中右侧面板默认隐藏
- 没有全局可见的导出入口

### 修复方案

#### 1. 在 TopBar 添加全局导出按钮
在顶部栏的右侧添加一个明显的"Export"按钮，与现有按钮（Randomize、Presets、Share）并列显示。点击后直接打开导出模态框。

**修改文件**: `components/TopBar.tsx`
- 添加 `Code2` 图标导入
- 添加导出按钮，与现有按钮组并列
- 点击打开 `ExportModal` 组件

#### 2. 简化移动端标签页导航
在移动端底部导航栏添加 Code 标签页的快捷入口，让用户更容易切换到代码视图。

**修改文件**: `components/MainLayout.tsx`
- 移动端标签导航已有 Code 按钮，只需确保样式醒目

#### 3. 验证现有组件集成
确保 `ExportModal` 组件能被正确调用，无需依赖 `OutputPanel` 内部的 `ExportButtons`。

**修改文件**: `components/output/ExportModal.tsx`
- 确保组件可独立使用
- 添加 portal 支持以确保 z-index 正确
