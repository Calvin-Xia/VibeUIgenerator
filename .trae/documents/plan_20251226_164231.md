## Card 元素字体粗细无效问题修复方案

### 问题分析

通过代码检查发现两个问题：

1. **getCardStyles 缺少 fontWeight**（第 484-496 行）：
   ```typescript
   const styles: React.CSSProperties = {
     fontFamily: theme.typography.fontFamily,
     fontSize: theme.typography.fontSize,
     // fontWeight 缺失！
     color: theme.palette.text,
     ...
   }
   ```

2. **CardPreview 内部内容没有应用字体样式**：
   - 卡片内部的占位 div（模拟标题和内容）没有 font-weight 样式
   - 它们只是用 colored divs 模拟文本，没有实际应用 typography 样式

### 修复方案

1. **修改 getCardStyles**：
   - 添加 `fontWeight: theme.typography.fontWeight`

2. **修改 CardPreview**：
   - 为卡片内部的内容元素添加 `fontWeight` 样式
   - 确保所有文本内容都应用主题的字体粗细设置

### 修改文件
- `lib/generator/index.ts`
- `components/preview/CardPreview.tsx`

### 预期效果
- Typography Font Weight 滑动条同时影响 button 和 card 元素
- Card 内的占位文本也会应用正确的字体粗细
- 两种元素的字体粗细保持一致
